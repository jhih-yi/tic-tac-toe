# tic-tac-toe

井字遊戲,共有三種模式  
1.雙人模式(two player mode)  
2.普通電腦模式(stupid computer mode) : 程式隨機挑選空位  
3.ai電腦模式(ai computer mode) : 利用minimax選擇最佳空位
